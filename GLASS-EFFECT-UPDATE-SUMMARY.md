## 🎨 CSS 技术细节

### Tailwind 透明度语法

```css
bg-primary/20  →  rgba(37, 98, 194, 0.2)
bg-primary/30  →  rgba(37, 98, 194, 0.3)
bg-primary/40  →  rgba(37, 98, 194, 0.4)
```

### 毛玻璃效果

```css
backdrop-blur-sm   →  backdrop-filter: blur(4px)
backdrop-blur-md   →  backdrop-filter: blur(8px)
backdrop-blur-lg   →  backdrop-filter: blur(16px)
```

### 完整的玻璃罩效果

```tsx
<div 
  className="backdrop-blur-md rounded-2xl p-4 border-2 border-white/40"
  style={{
    background: 'rgba(255, 255, 255, 0.98)',
    boxShadow: '0 8px 32px rgba(0, 0, 0, 0.15)'
  }}
>
```

- 开发服务器未重启

```bash

# 2. 清除缓存

# 3. 重新启动
npm run dev
```

- 图片路径错误
- 图片文件不存在

**解决方案**:
```bash

ls -lh public/pgc_mba_landing_images/hero-campus-billboard.webp
```
