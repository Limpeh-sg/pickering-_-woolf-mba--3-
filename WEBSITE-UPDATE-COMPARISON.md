### 9.0 蓝色玻璃罩效果专项分析 ⭐

**这是本地版本的核心视觉特色！**

#### 📱 实现细节

**1. Hero 区域主背景玻璃罩**

**桌面端代码**:
```tsx
{/* 蓝色玻璃罩遮罩 - 20% 不透明度 */}
<div className="absolute inset-0 bg-primary/20 pointer-events-none" />
```

**移动端代码**:
```tsx
{/* 移动端背景图 */}
<img src="/pgc_mba_landing_images/hero-campus-billboard.webp" />
{/* 更深的蓝色玻璃罩 - 35% 不透明度 */}
<div className="absolute inset-0 bg-primary/35" />
```

**效果说明**:
- 🎨 **颜色**: 使用品牌主色 `primary` (蓝色 #2562C2)
- 📊 **透明度**: 桌面 20% / 移动 35%
- 🎯 **目的**: 
  - 统一视觉风格
  - 提升文字可读性
  - 强化品牌识别
  - 营造专业氛围

**2. 认证卡片毛玻璃效果**

```tsx
<div 
  className="backdrop-blur-md rounded-2xl p-4 w-36 border-2 border-white/40"
  style={{
    background: 'rgba(255, 255, 255, 0.98)',
    boxShadow: '0 8px 32px rgba(0, 0, 0, 0.15)'
  }}
>
```

**技术要点**:
- `backdrop-blur-md`: 中等强度背景模糊 (8px)
- `rgba(255, 255, 255, 0.98)`: 98% 不透明白色
- `border-white/40`: 40% 不透明白色边框
- 深度阴影创造悬浮感

**3. 其他玻璃罩应用**

| 位置 | 效果 | 代码 |
|------|------|------|
| 顶部导航栏 | 半透明白色 + 模糊 | `bg-white/80 backdrop-blur-md` |
| Flash Banner | 白色半透明 | `bg-white border-b` |
| 学员评价卡片 | 白色10%透明 + 模糊 | `bg-white/10 backdrop-blur-sm` |
| 新加坡研学区 | 蓝色40%遮罩 | `bg-primary/40` |

#### 💡 设计理念

**玻璃态设计 (Glassmorphism)** 的核心原则:
1. ✅ **透明度**: 使用半透明背景
2. ✅ **模糊效果**: backdrop-filter: blur()
3. ✅ **鲜明边框**: 半透明边框增强边界
4. ✅ **多层叠加**: 创造深度和层次
5. ✅ **光影效果**: 阴影和高光模拟玻璃质感

**为什么选择蓝色玻璃罩？**
- 🎨 **品牌一致性**: 蓝色是 Pickering 的主色调
- 📚 **教育行业惯例**: 蓝色代表专业、信任、智慧
- 🌏 **国际化感觉**: 蓝色具有全球通用性
- 👁️ **视觉舒适**: 蓝色对眼睛友好，适合长时间浏览

#### 🎨 蓝色玻璃罩效果详解

**本地版本的蓝色玻璃罩实现**:

**桌面端 (Desktop)**:
```css
/* 背景图片上方的蓝色半透明遮罩 */
.absolute.inset-0.bg-primary/20
```
- 使用 `bg-primary/20` = 蓝色背景 + 20% 不透明度
- 覆盖在校园图片上方
- 创造出**轻微的蓝色玻璃罩效果**
- 让图片更柔和，文字更易读

**移动端 (Mobile)**:
```css
/* 更强的蓝色毛玻璃效果 */
.absolute.inset-0.bg-primary/35
```
- 使用 `bg-primary/35` = 蓝色背景 + 35% 不透明度
- **比桌面端更深的蓝色遮罩**
- 确保移动端文字清晰可读
- 营造更强的品牌色氛围

**认证卡片的玻璃罩效果**:
```css
/* 白色毛玻璃卡片 */
background: rgba(255, 255, 255, 0.98)
backdrop-blur-md
border: 2px solid rgba(255, 255, 255, 0.4)
box-shadow: 0 8px 32px rgba(0, 0, 0, 0.15)
```
- 98% 不透明的白色背景
- `backdrop-blur-md` = 中等强度的背景模糊
- 半透明白色边框
- 深度阴影增强立体感
- 创造出**高级的毛玻璃质感**

**视觉效果对比**:

| 效果 | 线上版本 | 本地版本 |
|------|---------|---------|
| 背景遮罩 | 无或很淡 | 蓝色 20-35% 不透明度 |
| 卡片质感 | 纯白色实心 | 毛玻璃半透明 |
| 视觉层次 | 较平面 | 立体分层 |
| 品牌感 | 一般 | 强烈 |
| 现代感 | 传统 | 高级 |
