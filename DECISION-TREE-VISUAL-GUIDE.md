## 🎯 评分示例

### 示例1：高意向申请者（Hot Lead）

```
路径：
Level 1: 申请材料 (+18)
Level 2: 学位、成绩单、简历、护照基本齐了 (+30)
Level 3: 我准备近期申请 (+24)
Level 4: 我准备开始申请 (+30)
Level 5: 提交表单 (+30)

总分：5 (打开) + 18 + 30 + 24 + 30 + 30 = 137分
等级：Hot Lead (≥70)
建议：立即联系（30分钟内），协助提交申请材料
```

### 示例2：费用敏感用户（Warm Lead）

```
路径：
Level 1: 费用 (+12)
Level 2: 奖学金可能性与类别 (+15)
Level 3: 希望获得个性化费用/奖学金预估 (+18)
Level 4: WhatsApp发送摘要 (+20)
Level 5: 提交表单 (+30)

总分：5 (打开) + 12 + 15 + 18 + 20 + 30 = 100分
等级：Hot Lead (≥70)
建议：24小时内通过WhatsApp跟进，发送费用拆分和奖学金说明
```

### 示例3：信息收集者（Cold Lead）

```
路径：
Level 1: 手册 (+5)
Level 2: 发送完整手册 (+5)
Level 3: 我想先下载资料 (+5)
Level 4: 先发手册，暂时不电话 (+5)
Level 5: 提交表单 (+30)

总分：5 (打开) + 5 + 5 + 5 + 5 + 30 = 55分
等级：Warm Lead (40-69)
建议：24小时内发送定制资料，3天后再次跟进
```

## 🎯 使用场景

### 场景1：快速申请者
```
用户意图：想尽快申请
推荐路径：申请材料 → 基本齐了 → 准备近期申请 → 准备开始申请
预期分数：≥100分
预期等级：Hot Lead
顾问动作：30分钟内联系，协助提交
```

### 场景2：费用顾虑者
```
用户意图：担心费用太高
推荐路径：费用 → 奖学金 → 个性化预估 → WhatsApp
预期分数：60-80分
预期等级：Warm/Hot Lead
顾问动作：24小时内发送费用拆分和奖学金说明
```

### 场景3：学历认可顾虑者
```
用户意图：担心学历不被认可
推荐路径：学历认可 → Woolf认证 → 书面说明 → 谨慎解释
预期分数：40-60分
预期等级：Warm Lead
顾问动作：24小时内发送谨慎说明，适合转发
```
