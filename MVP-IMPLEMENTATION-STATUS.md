## 📝 注意事项

1. **不是聊天机器人**：这是决策树系统，不支持自由对话
2. **严格5级**：不能超过5级，遇到分流要压平
3. **谨慎文案**：特别是学历认可路径，不能过度承诺
4. **数据驱动**：每个选择都要记录、评分、打标签
5. **顾问友好**：生成的线索要让顾问知道怎么跟进

## 🎨 设计原则

1. **现代玻璃感**：毛玻璃效果，白色半透明
2. **无AI感**：去掉所有emoji和AI元素
3. **专业简洁**：短句引导，长文变成"查看详情"
4. **移动优先**：PC右下角，移动端半屏
5. **进度可见**：显示1/5到5/5进度
