## 🎯 核心原则（已遵循）

1. ✅ **严格5级**：不超过5级，不无限聊天
2. ✅ **决策树而非聊天**：用户选择选项，不是自由对话
3. ✅ **压缩转化**：从模糊兴趣到结构化线索
4. ✅ **数据驱动**：每个选择都记录、打分、打标签
5. ✅ **谨慎文案**：特别是学历认可路径，不过度承诺
6. ✅ **短句引导**：简洁明了，不堆砌长文
7. ✅ **顾问友好**：生成的线索包含明确的跟进建议

## 🚀 使用方法

### 用户端
1. 点击右下角浮动按钮打开助手
2. 回答5个级别的问题（每级选择一个选项）
3. 填写联系表单
4. 提交并查看摘要
5. 可选：复制或下载Payload

### 顾问端
1. 在Lark中接收线索通知
2. 查看线索等级（Cold/Warm/Hot）
3. 查看用户完整选择路径
4. 查看自动生成的跟进建议
5. 根据建议执行跟进动作

## 📋 顾问跟进SOP

### Hot Lead（分数≥70）
**特征**：
- 选择"近期申请"
- 选择"开始申请"
- 选择"电话/线上咨询"
- 选择"申请材料基本齐了"

**动作**：
- ⏰ 30分钟内联系
- 📱 优先WhatsApp/电话
- 📝 协助提交申请材料
- 📅 确认入学批次

### Warm Lead（分数40-69）
**特征**：
- 关注费用、奖学金、学历认可
- 选择发送资料
- 尚未明确申请时间

**动作**：
- ⏰ 24小时内跟进
- 📧 发送定制资料
- 🔄 3天后再次跟进
- 🎓 邀请参加线上说明会

### Cold Lead（分数<40）
**特征**：
- 只是下载资料
- 明年再规划
- 不愿电话

**动作**：
- 📚 进入nurture list
- 📄 发送手册
- 🔄 7天后轻度跟进
- 🎯 加入再营销广告池

## 📈 数据分析建议

### 关键指标
1. **转化率**：打开助手 → 完成第1级 → 完成第5级
2. **路径分布**：哪个第1级选项最受欢迎
3. **线索质量**：Hot/Warm/Cold比例
4. **掉线点**：用户在哪一级最容易离开
5. **标签分布**：最常见的用户顾虑和兴趣

### 优化方向
1. **高掉线路径**：简化选项或调整文案
2. **低转化路径**：增加引导或激励
3. **高价值标签**：针对性广告投放
4. **顾问反馈**：调整跟进建议准确性
